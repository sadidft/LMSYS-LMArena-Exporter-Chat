# LMArena Chat Exporter

JavaScript utility to export chat conversations from LMArena to Markdown format. This script runs in browser console and automatically detects user and assistant messages, then copies the result to clipboard.

## Features

- Export LMArena chat to clean Markdown format
- Automatic user/assistant message detection based on order
- Automatic content cleaning (whitespace, excessive line breaks)
- Auto-copy to clipboard
- Manual copy fallback if clipboard fails
- Message order reverse option
- Support multiple LMArena DOM selectors

## Quick Start

1. Open your LMArena chat page
2. Press `F12` to open browser console
3. Paste the script and press Enter
4. Follow the prompts
5. Paste result in your text editor
